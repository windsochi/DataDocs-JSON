== Как это работает

* удалённое приложение отправляет запрос вида: http://localhost:3000/docs.json?start=2002-01-01&finish=2002-12-31

* метод контроллёра index получает запрос

* обрабатывает его, используя интервальное условие: <tt>Doc.where(date_of_issue: params[:start]..params[:finish])</tt>

* отдаёт ответ в виде JSON
